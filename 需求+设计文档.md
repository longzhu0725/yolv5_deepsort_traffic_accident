## 第一部分：需求规格

---

### 1. 项目概述

| 条目 | 内容 |
|------|------|
| 项目名称 | 基于 YOLOv5 与 DeepSORT 的交通事故实时定位与动态警示系统 |
| 核心思路 | YOLOv5 逐帧检测车辆/行人 → DeepSORT 跨帧跟踪并维护轨迹 → 分析轨迹异常（碰撞+速度骤变）判定事故 → 映射地理坐标 → Web 端实时告警展示 |
| 技术关键词 | YOLOv5、DeepSORT、Flask、Vue 3、WebSocket、Leaflet、ECharts |
| 演示形态 | 浏览器打开 Web 页面，观看实时标注视频，事故发生时弹窗+声音+地图标注 |

### 2. 用户角色（极简）

| 角色 | 说明 |
|------|------|
| 管理员 (admin) | 唯一角色，登录后可使用所有功能。系统预置一个账户 `admin / admin123` |

### 8. 线程模型

```
主线程 (Flask)
  ├── HTTP 请求处理
  ├── SocketIO 事件处理
  └── MJPEG 视频流响应（从 frame_buffer 读取）

工作线程 (每个运行中的摄像头一个)
  └── Pipeline._run_loop()
      读帧 → YOLOv5检测 → DeepSORT跟踪 → 事故判定
      → 标注帧写入 frame_buffer (加锁)
      → 有事故时通过 socketio.emit() 推送

共享数据（线程间）:
  frame_buffer: dict  {camera_id: numpy.ndarray}  ← threading.Lock 保护
```

```sql
    id              INTEGER PRIMARY KEY AUTOINCREMENT,

    camera_id       INTEGER      NOT NULL,

    -- 时间
    trigger_time    DATETIME     NOT NULL,

    -- 位置
    pixel_x         INTEGER      DEFAULT NULL,
    pixel_y         INTEGER      DEFAULT NULL,
    geo_lat         REAL         DEFAULT NULL,
    geo_lng         REAL         DEFAULT NULL,
    address         VARCHAR(300) DEFAULT NULL,

    -- 评分
    severity        VARCHAR(20)  DEFAULT 'normal',       -- normal/serious/critical
    score_total     REAL         DEFAULT 0,
    score_collision REAL         DEFAULT 0,
    score_speed     REAL         DEFAULT 0,

    -- 涉事
    involved_ids    TEXT         DEFAULT NULL,            -- JSON: [12, 34]
    involved_count  INTEGER      DEFAULT 0,

    -- 媒体

    FOREIGN KEY (camera_id) REFERENCES cameras(id)
```

##### 11.6.1 YOLOv5 检测器（`app/core/detector.py`）

```python
"""
类: YOLOv5Detector

构造参数:
    weights_path: str
    conf: float = 0.5
    iou: float = 0.45
    img_size: int = 640
    device: str = ''
    target_classes: dict = None

初始化逻辑:
    1. 将项目根目录下的 yolov5/ 加入 sys.path
    2. model = torch.hub.load('./yolov5', 'custom', path=weights_path, source='local')
       或者使用: model = torch.hub.load('ultralytics/yolov5', 'custom', path=weights_path)
    3. model.conf = conf
    4. model.iou = iou
    5. 如果 target_classes 不为空: model.classes = list(target_classes.keys())
    6. 用一张空图预热: model(np.zeros((img_size, img_size, 3), dtype=np.uint8))

方法:

def detect(self, frame: np.ndarray) -> list:
    '''
    返回: [
        {'bbox': [x1,y1,x2,y2], 'confidence': float, 'class_id': int, 'class_name': str},
        ...
    ]
    
    实现:
    results = self.model(frame, size=self.img_size)
    preds = results.xyxy[0].cpu().numpy()  # (N, 6) [x1,y1,x2,y2,conf,cls]
    遍历 preds，构建返回列表
    '''

def detect_for_tracker(self, frame: np.ndarray) -> tuple:
```

##### 11.6.3 事故检测器（`app/core/accident_detector.py`）⭐ 核心算法

```python
"""
数据类: AccidentEvent
    involved_track_ids: list[int]    # 涉事目标 ID
    pixel_position: tuple (x, y)     # 事故像素位置 (两目标中心的中点)
    score_total: float               # 综合得分
    score_collision: float           # 碰撞分
    score_speed: float               # 速度分
    severity: str                    # 'normal'(0.6~0.75) / 'serious'(0.75~0.9) / 'critical'(>0.9)
    trigger_time: float              # time.time()


类: AccidentDetector

构造参数:
    w_collision: float = 0.5         # 碰撞权重
    w_speed: float = 0.5             # 速度权重
    threshold: float = 0.6           # 综合阈值
    collision_iou: float = 0.3       # 碰撞 IoU 阈值
    collision_frames: int = 3        # 碰撞持续帧数阈值
    speed_drop_pct: float = 0.7      # 速度骤降百分比
    speed_window: int = 5            # 速度窗口帧数
    cooldown_seconds: float = 30     # 冷却时间
    cooldown_distance: float = 100   # 冷却距离(像素)

内部状态:
    collision_counter: dict          # {(id_a, id_b): 连续碰撞帧数}
    recent_alerts: list              # [(timestamp, x, y), ...] 冷却用


方法:

def analyze(self, tracks: list[TrackInfo]) -> list[AccidentEvent]:
    '''
    核心方法 - 分析所有轨迹，返回检测到的事故列表

    算法流程:
    accidents = []
    
    # 两两遍历所有轨迹
    for i in range(len(tracks)):
        for j in range(i+1, len(tracks)):
            track_a = tracks[i]
            track_b = tracks[j]
            
            # 因子1: 碰撞检测
            s_collision = self._check_collision(track_a, track_b)
            
            # 因子2: 速度骤变 (取两者中更严重的)
            s_speed = max(
                self._check_speed_anomaly(track_a),
                self._check_speed_anomaly(track_b)
            )
            
            # 综合评分
            score = self.w_collision * s_collision + self.w_speed * s_speed
            
            # 超阈值 + 通过冷却检查 → 触发事故
            if score >= self.threshold:
                mid_x = (track_a.center[0] + track_b.center[0]) // 2
                mid_y = (track_a.center[1] + track_b.center[1]) // 2
                
                if not self._is_cooling(mid_x, mid_y):
                    severity = 'critical' if score >= 0.9 else 'serious' if score >= 0.75 else 'normal'
                    event = AccidentEvent(
                        involved_track_ids=[track_a.track_id, track_b.track_id],
                        pixel_position=(mid_x, mid_y),
                        score_total=round(score, 3),
                        score_collision=round(s_collision, 3),
                        score_speed=round(s_speed, 3),
                        severity=severity,
                        trigger_time=time.time()
                    )
                    accidents.append(event)
                    self.recent_alerts.append((time.time(), mid_x, mid_y))
    
    return accidents
    '''


def _check_collision(self, track_a, track_b) -> float:
    '''
    碰撞检测因子

    1. iou = _compute_iou(track_a.bbox, track_b.bbox)
    2. key = (min(a.id, b.id), max(a.id, b.id))
    3. if iou > self.collision_iou:
         collision_counter[key] += 1
         if collision_counter[key] >= self.collision_frames:
             return min(1.0, iou * 1.5)   # 放大IoU作为分数
       else:
         collision_counter[key] = 0
    4. return 0.0
    '''


def _check_speed_anomaly(self, track) -> float:
    '''
    速度骤变因子

    1. history = list(track.velocity_history)
    2. if len(history) < self.speed_window: return 0.0
    3. recent = history[-self.speed_window:]
    4. first_half = recent[:len(recent)//2]   # 前半段
    5. second_half = recent[len(recent)//2:]  # 后半段
    6. max_v = max(first_half) if first_half else 0
    7. min_v = min(second_half) if second_half else 0
    8. if max_v > 5.0:  # 排除本来就很慢的
         drop_ratio = (max_v - min_v) / max_v
         if drop_ratio >= self.speed_drop_pct:
             return min(1.0, drop_ratio)
    9. return 0.0
    '''


def _is_cooling(self, x, y) -> bool:
    '''
    冷却检查 - 防止同一位置重复报警

    1. now = time.time()
    2. 清理过期记录: self.recent_alerts = [(t,rx,ry) for ... if now-t < cooldown_seconds]
    3. 遍历 recent_alerts:
         if 距离((x,y), (rx,ry)) < cooldown_distance:
             return True
    4. return False
    '''


def _compute_iou(self, box_a, box_b) -> float:
    '''标准 IoU 计算，box 格式 [x1,y1,x2,y2]'''


def reset(self):
    '''清空 collision_counter 和 recent_alerts'''
"""
```

##### 11.6.6 处理流水线（`app/core/pipeline.py`）⭐ 核心集成

```python
"""
类: ProcessingPipeline

构造参数:
    camera_id: int
    camera_config: dict           # 从 Camera.to_dict() 获取
    detector: YOLOv5Detector      # 共享实例
    socketio: SocketIO            # 推送用
    app: Flask                    # 访问数据库上下文
    frame_buffer: dict            # 共享帧缓冲
    buffer_lock: threading.Lock   # 缓冲锁

内部属性:
    self.tracker: DeepSORTTracker          # 每路独立
    self.accident_detector: AccidentDetector
    self.coordinate_mapper: CoordinateMapper or None
    self.is_running: bool
    self.thread: threading.Thread or None
    self.fps: float
    self.cap: cv2.VideoCapture or None

方法:

def start(self):
    '''
    1. self.is_running = True
    2. self.thread = Thread(target=self._run, daemon=True)
    3. self.thread.start()
    '''

def stop(self):
    '''
    1. self.is_running = False
    2. if self.thread: self.thread.join(timeout=5)
    3. if self.cap: self.cap.release()
    '''

def _run(self):
    '''
    核心循环，在工作线程中执行:
    
    # 打开视频
    self.cap = cv2.VideoCapture(self.camera_config['stream_url'])
    if not self.cap.isOpened():
        self._set_camera_status('error')
        return
    
    self._set_camera_status('online')
    frame_count = 0
    fps_timer = time.time()
    
    while self.is_running:
        try:
            ret, frame = self.cap.read()
            if not ret:
                # 视频结束 → 循环播放 (演示用)
                self.cap.set(cv2.CAP_PROP_POS_FRAMES, 0)
                continue
            
            # 1. YOLOv5 检测
            bboxes, confs, cls_ids = self.detector.detect_for_tracker(frame)
            
            # 2. DeepSORT 跟踪
            tracks = self.tracker.update(bboxes, confs, cls_ids, frame)
            
            # 3. 事故检测
            accidents = self.accident_detector.analyze(tracks)
            
            # 4. 处理事故
            for acc in accidents:
                self._handle_accident(acc, frame)
            
            # 5. 帧标注
            annotated = FrameAnnotator.annotate(
                frame, tracks, accidents, 
                self.fps, len(tracks), self.camera_config['name']
            )
            
            # 6. 写入共享缓冲
            with self.buffer_lock:
                self.frame_buffer[self.camera_id] = annotated
            
            # 7. 计算 FPS
            frame_count += 1
            if frame_count % 30 == 0:
                elapsed = time.time() - fps_timer
                self.fps = round(30.0 / elapsed, 1) if elapsed > 0 else 0
                fps_timer = time.time()
        
        except Exception as e:
            print(f'[Pipeline-{self.camera_id}] Error: {e}')
            import traceback; traceback.print_exc()
            continue
    
    if self.cap: self.cap.release()
    self._set_camera_status('offline')
    '''

def _handle_accident(self, accident: AccidentEvent, frame):
    '''
    处理一次事故事件:
    
    with self.app.app_context():
        # 1. 保存截图
        filename = f'acc_{self.camera_id}_{int(accident.trigger_time)}.jpg'
        filepath = os.path.join(Config.SNAPSHOT_FOLDER, filename)
        # 先标注再保存
        annotated = FrameAnnotator.draw_accident_marker(frame.copy(), accident)
        cv2.imwrite(filepath, annotated)
        
        # 2. 坐标映射 (如果有标定数据)
        geo_lat, geo_lng, address = None, None, None
        if self.coordinate_mapper:
            result = self.coordinate_mapper.pixel_to_geo(
                accident.pixel_position[0], accident.pixel_position[1])
            if result:
                geo_lng, geo_lat = result
                address = f'{geo_lng:.5f}, {geo_lat:.5f}'  # 简化地址
        
        # 3. 生成事故编号
        from datetime import datetime
        today = datetime.now().strftime('%Y%m%d')
        count = Accident.query.filter(
            Accident.accident_no.like(f'ACC-{today}-%')).count()
        accident_no = f'ACC-{today}-{count+1:03d}'
        
        # 4. 存入数据库
        record = Accident(
            accident_no=accident_no,
            camera_id=self.camera_id,
            trigger_time=datetime.fromtimestamp(accident.trigger_time),
            pixel_x=accident.pixel_position[0],
            pixel_y=accident.pixel_position[1],
            geo_lat=geo_lat, geo_lng=geo_lng, address=address,
            severity=accident.severity,
            score_total=accident.score_total,
            score_collision=accident.score_collision,
            score_speed=accident.score_speed,
            involved_ids=json.dumps(accident.involved_track_ids),
            involved_count=len(accident.involved_track_ids),
            snapshot_path=f'snapshots/{filename}',
            status='pending'
        )
        db.session.add(record)
        db.session.commit()
        
        # 5. WebSocket 推送告警
        self.socketio.emit('accident_alert', {
            'accident_id': record.id,
            'accident_no': record.accident_no,
            'camera_id': self.camera_id,
            'camera_name': self.camera_config['name'],
            'trigger_time': record.trigger_time.strftime('%Y-%m-%d %H:%M:%S'),
            'severity': accident.severity,
            'score_total': accident.score_total,
            'geo_lat': geo_lat, 'geo_lng': geo_lng,
            'address': address,
            'snapshot_url': f'/static/snapshots/{filename}',
            'message': f'检测到{accident.severity}级交通事故'
        })
    '''

def _set_camera_status(self, status):
    '''
    with self.app.app_context():
        camera = Camera.query.get(self.camera_id)
        if camera:
            camera.status = status
            db.session.commit()
    '''
"""
```
